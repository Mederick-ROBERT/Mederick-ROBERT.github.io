Aujourd-hui je me lance dans le coding afin d'étendre mes compétences dans ce milieu de l'informatique. A terme, je souhaite pouvoir être compétent dans tous domaines informatique, que ce soit le Hardware, le code ou encore la sécurité et le réseau.
